# Contributing to Aero Framework

Thank you for your interest in improving Aero! Here are the guidelines for contributing to this project.

## Reporting Bugs
If you encounter an issue, please open a GitHub Issue and include:
1. A clear description of the expected vs. actual behavior.
2. Consistent reproduction steps.
3. Relevant error dumps or console logs from the FiveM server.

## Code Contributions
1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/AmazingFeature`).
3. Implement your changes, ensuring the solution builds successfully without errors in JetBrains Rider.
4. Commit your changes and open a Pull Request against the `main` branch.
